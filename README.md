# ESXI Terraform VM Provisioner

An automated terraform configuration for batch deployment of Ubuntu machines on ESXi 

---

## Prerequisites

* [Terraform](https://www.terraform.io/) >= 1.0
* **VMware ESXi** hypervisor
* Ubuntu Server OVF/OVA image

---

## Quick Start

### 1.Clone the Repository

```bash
 gitclone https://github.com/KristianCheydy/esxi_terraform_nodes.git
 cd esxi_terraform_nodes
```
